Template for Developers and API Designers

Description

This extension can be used as a template or starting point when developing new extensions.

Note: If you want to enable this extension. You need to create a database table, because the model relies on it:

CREATE TABLE IF NOT EXISTS `dev_items` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `name` varchar(1000) NOT NULL,
  `description` varchar(1024) NOT NULL,
  PRIMARY KEY (`id`)
) ENGINE=InnoDB;

Endpoints

Endpoint Verb Parameters Description
/template/items GET Retrieves all items.
/template/items/:item_id GET Retrieves the item with :item_id.
/template/items POST $a_data['name'], $a_data['description'] Creates a new item.
/template/items/:item_id PUT $a_data['name'], $a_data['description'] Updates item with :item_id.
/template/items/:item_id DELETE Deletes the item with :item_id.
/template/items DELETE Deletes all items.

Examples

Gets a list of items
curl -X GET http://localhost/restplugin.php/template/items
